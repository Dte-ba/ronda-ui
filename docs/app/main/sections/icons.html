Icons

Los iconos pueden ser utilizados en cualquier tipo de elementos usando las clases .ri.ri-icon-name, no obstante aconsejamos utilizar el tag i

Tamaños de iconos

Los tamaños de los icons puede ser modificados agregando clases .ri-6x, .ri-5x, .ri-4x, .ri-3x, .ri-2x respectivamente. Ejemplo utilizando .ri.ri-herramienta
